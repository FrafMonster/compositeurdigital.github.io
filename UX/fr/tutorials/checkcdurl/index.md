## Vérifier un lien dans le Compositeur Digital UX

### Prérequis

Votre PC doit être équipé de Windows 10 et vous devez avoir le logiciel !Compositeur Digital UX d'installé sur votre poste. Reportez-vous à cette rubrique pour voir comment l'installer.

### Exécutez le lien

En utilisant l'explorateur de fichier, faîtes un double-click sur votre fichier `.cdurl`. Une fenêtre s'ouvre et vous demande avec quelle application vous souhaitez ouvrir ce lien.
Sélectionner Compositeur Digital UX.

Le Compositeur Digital UX se lance, et ouvre une vue web. Si vous êtes équipé d'un PC à écran tactile, essayez d'interagir avec la vue Web. Certains sites n'offrent pas un support tactile, l'expérience proposée dans le Compositeur Digital UX s'avère donc décevante.
